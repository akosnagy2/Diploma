%----------------------------------------------------------------------------
\chapter{Összegzés}
%----------------------------------------------------------------------------

Összegzésképpen elmondhatjuk, hogy a diplomatervem célkitûzéseit sikerült elérnem:

\begin{itemize}
\item Megismertem a robotikában alkalmazott pályatervezési megközelítéseket.
\item  A pályatervezõ algoritmusok közül egy módszert részletesen is megvizsgáltam, valamint implementáltam.
\item  Kifejlesztettem egy a pályatervezéshez kapcsolódó idõparaméterezési, és egy pályakövetõ eljárást.
\item Megismertem a V-REP robotszimulátor programot.
\item Elkészítettem a robotszimulátorban a differenciális robot modelljét.
\item Az algoritmusok mûködését szimulátorban sikerült igazolnom.
\item A szimulációnak megfelelõ eredményeket kaptam valós roboton történõ vizsgálatok során is.
\end{itemize}
 
Úgy látom, a jövõben a következõ fejlesztésekkel lehetne továbblépni a pályatervezés és mozgásirányítás témában: 

\begin{itemize}
\item További pályatervezõ módszerek implementálása.
\item Pályatervezõ algoritmusok összehasonlítása szimulátorban végzett mérések alapján.
\item Kinematikai robotmodell lecserélése dinamikai modellre.
\item Idõparaméterezésnél további korlátozások figyelembe vétele.
\item Az algoritmusok tesztelése, elsõsorban a tapasztalt mechanikai problémák miatt, más differenciális, valós roboton.
\item Keretrendszer fejlesztése, elsõsorban a könnyebb használat érdekében.
\item Valós robot irányítószoftverének továbbfejlesztése, a bemutatott algoritmusok alkalmazása az Eurobot 2015 robotversenyen.
\end{itemize}
